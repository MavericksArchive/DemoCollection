situation: Summary: Cisco has provided an update on the ongoing investigation into the exploitation of the web UI feature in Cisco IOS XE Software. The attackers exploited two previously unknown issues, CVE-2023-20198 and CVE-2023-20273. CVE-2023-20198 has a CVSS score of 10.0, while CVE-2023-20273 has a CVSS score of 7.2. The vulnerabilities can be detected by checking if the HTTP Server feature is enabled through the "show running-config | include ip http server|secure|active" command. Cisco recommends disabling the HTTP Server feature or restricting its access to trusted source addresses. Fixed software releases are available for affected versions of Cisco IOS XE Software.
